# ResearchStack Website

Ultra-modern, sleek website for ResearchStack - Your Research Management Companion

## Files

- `index.html` - Main landing page with features, pricing, and download sections
- `privacy.html` - Comprehensive privacy policy
- `styles.css` - Modern CSS with gradients, animations, and responsive design
- `script.js` - Interactive JavaScript for smooth scrolling and animations

## Responsive Design

The website is fully responsive with breakpoints at:
- Desktop: 1024px+
- Tablet: 768px - 1024px
- Mobile: < 768px

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- iOS Safari (for mobile viewing)
- Supports CSS Grid, Flexbox, and modern CSS features

## Setup

1. Create an `images/` folder in the WebPage directory
2. Add your screenshots and images with the filenames listed above
3. Optionally create a `videos/` folder for video demos
4. Open `index.html` in a web browser

## Customization

### Colors
Edit CSS variables in `styles.css` under `:root` selector

### Content
Update text in `index.html` and `privacy.html`

### Animations
Modify animation keyframes in `styles.css` and `script.js`

## Privacy-First Messaging

## SEO Considerations

- Semantic HTML5 markup
- Descriptive meta tags (add to `<head>`)
- Alt text for images (add when images are available)
- Clean URL structure
- Fast loading (minimal dependencies)

## Performance

- No external dependencies (except images)
- Vanilla JavaScript (no frameworks)
- Optimized CSS (no bloat)
- Lazy loading ready for images
- Minimal HTTP requests
